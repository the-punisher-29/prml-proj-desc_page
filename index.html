Abstract

With the ever-increasing amount of movie content available, providing personalized recommendations to users has become a crucial challenge. This project presents a comprehensive movie recommendation system that employs multiple machine learning models to cater to diverse user preferences and provide accurate recommendations. The system leverages the Movies Lens dataset containing 100k ratings to train and evaluate the models.

The recommendation system utilizes four main models: K-Nearest Neighbors (KNN), Artificial Neural Networks (ANN), Support Vector Machines (SVM), and Gaussian Mixture Models (GMM). The KNN model finds similar users based on movie preferences, while the ANN model predicts user preferences by capturing non-linear relationships between movie features and user feedback. The SVM model classifies users into different preference groups, enabling targeted recommendations, and the GMM clusters users based on their movie-watching patterns and preferences.

Additionally, a Linear Regression model is employed to predict movies based on user IDs, providing a complementary recommendation approach. By combining these diverse models and leveraging various input features such as user IDs, movie names, genres, ratings, and user feedback, the system offers a robust and personalized recommendation experience.

The proposed movie recommendation system demonstrates the power of machine learning techniques in addressing the challenges of content recommendation. By incorporating multiple models and leveraging different feature sets, the system aims to provide accurate and tailored recommendations, enhancing the overall user experience in the vast landscape of movie content.
